## Running
* main: $python main.py

## There is in the game:
* Save, load and edit maps
* Pause
* Move

## Controls
* Place blocks: Left Mouse
* Select build block: 0-3
* Editor on/off: E
* Load map: Ctrl+L
* Save map: Ctrl+S
* Scale map: -, +
* Move: WASD
* Exit: Esc
* Pause: P
